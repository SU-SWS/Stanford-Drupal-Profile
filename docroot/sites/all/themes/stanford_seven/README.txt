Stanford Seven Administration Theme for Drupal 7.x

-- SUMMARY -- 

Authors: Megan Erin Miller
This theme is for use as an administration theme and is based on Seven.

-- REQUIREMENTS --

This theme is intended for Drupal versions 7 only; it will not work with Drupal 6 or below. You will need Drupal's Seven theme as a base theme for this theme to operate properly.

-- INSTALLATION --

As an admin, go to Administer > Appearance and select Stanford Seven as the administration theme.
More detailed information on installing themes here: http://drupal.org/getting-started/install-contrib/themes
